## I. THE PHILOSOPHY

### The Core Truth
The About Me profile is not a form. It is not a questionnaire. It is not a one-time exercise.

**The About Me profile is a living, breathing digital representation of who you are.**

It evolves as you evolve. It deepens as you deepen. It learns from every brain dump, every chat, every focus session, every insight. It is the foundation upon which every Mindooo recommendation is built. Without it, the AI is generic. With it, the AI is personal.

### The Three Sources of Truth

| Source | How It Works | Example |
|---|---|---|
| **Direct Input** | You answer guided questions in the About Me module | "My peak energy hours are 9am-12pm" |
| **AI Extraction** | The AI mines your chronicles and chats for signals | "I notice you write about coding anxiety 3x per week" |
| **Behavioural Inference** | The system observes your usage patterns | "Your focus sessions are 40% longer on Tuesdays" |

### Why This Matters for Mo

Mo has dyslexia-type learning challenges and ADHD-type execution challenges. He cannot sustain long forms. He cannot remember what he wrote last week. He needs a system that:
- Breaks self-discovery into tiny, manageable pieces
- Remembers everything so he doesn't have to
- Updates itself from his natural behaviour
- Shows progress so he feels momentum, not overwhelm

**If the About Me profile works for Mo, it works for anyone.**

## IX. THE EVIDENCE & UPDATE RULES

### Rule 1: Auto-Save Everything
Every keystroke is saved. Never lose progress.

### Rule 2: Section Completion Calculation
Each section calculates its own completion based on answered questions:
- `completion = (answered_questions / total_questions) * 100`
- Updated in real-time
- Stored in `sectionX_completion` column

### Rule 3: Overall Completion Calculation
```javascript
function calculateOverallCompletion(profile) {
  const weights = {
    section1: 0.08,   // Current Situation
    section2: 0.10,   // Personality
    section3: 0.08,   // Energy
    section4: 0.08,   // Passions
    section5: 0.10,   // Values
    section6: 0.10,   // Ikigai
    section7: 0.08,   // Financial
    section8: 0.07,   // Relationships
    section9: 0.08,   // Dreams & Fears
    section10: 0.12,  // Blockers
    section11: 0.05,  // Health
    section12: 0.07,  // Cognitive
    section13: 0.05,  // Learning Style
    section14: 0.05   // Legacy
  }

  let total = 0
  for (const [section, weight] of Object.entries(weights)) {
    const sectionCompletion = profile[`${section}_completion`] || 0
    total += sectionCompletion * weight
  }

  return Math.round(total)
}
```

### Rule 4: AI-Extracted Updates
When the AI detects a profile signal in a brain dump or chat:
1. Flag the potential update
2. Present to user: "I noticed you mentioned [X]. Should I add this to your profile?"
3. User confirms → Update saved
4. User declines → Flag dismissed, not asked again

### Rule 5: Weekly Deepening Questions
Every week, the AI asks ONE question based on:
- Incomplete sections
- Recent activity patterns
- Gaps in self-knowledge
- Current blockers

### Rule 6: Version History
Every major update creates a snapshot:
- `profile_version` increments
- Old version stored in `self_model_snapshots` table
- User can view "How I've Changed" timeline
